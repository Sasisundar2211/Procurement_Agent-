# Prompt Libraries for Gemini API

This file contains a list of prompt libraries that can be used to generate prompts for the Gemini API.

- **Prompt-Engineering-Guide**: A collection of prompt engineering techniques and examples.
  - [GitHub](https://github.com/dair-ai/Prompt-Engineering-Guide)
- **LangChain**: A framework for developing applications powered by language models.
  - [GitHub](https://github.com/langchain-ai/langchain)
- **LlamaIndex**: A data framework for LLM applications to ingest, structure, and access private or domain-specific data.
  - [GitHub](https://github.com/run-llama/llama_index)
- **Semantic Kernel**: A lightweight SDK that lets you easily mix conventional programming languages with the latest in large language model (LLM) AI.
  - [GitHub](https://github.com/microsoft/semantic-kernel)
